Se puede utilizar para añadir audio y video sin utilizacion de flash u otro reproductor multimedia, se puede utilizar geolocalizacion que permite al sitio detectar la ubicacion de cada usuario que ingresa al sitio web, se pueden crear animaciones en 2D, tambien la API permite añadir eventos de teclado, mouse, etc.
